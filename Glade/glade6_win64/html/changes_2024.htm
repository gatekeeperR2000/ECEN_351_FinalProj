Glade Reference

Change Log 2024

Contents for 2023 can be found here

 

Version 6.0.1 release on 23/01/2024

- enhancement: Made zoom in/out smoother on Mac.

- bugfix: Opening a gzipped oasis file did not give an error. Now it does, as this is not supported.

- bugfix: Opening a library with the same name as existing library and choosing discard didnn't delete the existing library correctly.

- enhancement: Dark mode support - better colour choice for message window, LSW etc.

Version 6.0.2 release on 12/02/2024

- bugfix: Tweaked the size of import/export GDS2/OASIS dialogs for limited size laptop displays.

- bugfix: GDS mapping for existing libraries did not set up export in version 6 correctly.

- bugfix: GDS export warning messages not output.

- bugfix: Fix a crash if a design with MPPs had its techfile reloaded to update MPPs.

- bugfix: Export GDS2 with path style extend and non-halfwidth extensions exported incorrectly.

- bugfix: Fix a crash if run in non-graphics mode (-ng) and dark mode.

Version 6.0.3 release on 01/03/2024

- enhancement: multithread the KD tree.

- bugfix: Querying a polygon gave missing 4th arg to coord.

Version 6.0.4 release on 04/04/2024

- enhancement: Import Cadence techfile uses a new Skill parser that can evaluate basic Skill expressions.

- enhancement: Improved performance of LSW refresh when dealing with 100's of layers.

Version 6.0.5 release on 21/05/2024

- enhancement: Improved schematic autorouter obstacle modelling.

- enhancement: Moving an instance or a pin autoroutes connected wires in realtime.

Version 6.0.6 release on 01/06/2024

- enhancement: Create Via dialog allows creating via with no net info (the default).

- enhancement: Import Cadence techfile parses cdsViaDevice types. However, not all features are supported - if anyone can provide the documentation for cdsViaDevice them this could be fixed.

- enhancement: Import Cadence techfile parses streamLayers section.

Version 6.0.7 release on 17/06/2024

- bugfix: F8 did not toggle immediate move mode.

- enhancement: Added techfile layer attributes minDiagSpacing, minDiagWidth, minHoleArea.

- enhancement: Cence Skill techfile parser supports above attributes in techfile now.

- enhancement: Verify... DRC command supports minDiagSpacing, minDiagWidth, minHoleArea, minOverlap

- enhancement: minExt and minOvlp seperated and working as specified. Techfile keyword MINOVLP added.

- enhancement: DRC Verify... cmd has added checks.

Version 6.0.8 release on 24/06/2024

- enhancement: Techfile support for custom line styles (i.e. other than Qt default linestyles).

- enhancement: Stipple Editor dialog shows line styles graphically.

- bugfix: some stipple pattern sizes not expanded correctly to 32 bits.

Version 6.0.9 release on 04/07/2024

- enhancement: Skill file reader initial implementation.

Version 6.0.10 release on 06/09/2024

- enhancement: MIMCAP capacitors can be extracted with properties w (width of metal fingers), sp (space of metal fingers), lr (length of fingers) and nr (number of fingers).

- bugfix: Crash when closing attached lib.

- bugfix: spice3 run had extra blank in args which could cause simulation failure.

- bugfix: Fix issue when reading oasis without a techfile.

- bugfix: Allow -oasisout and -gdsout on cmd line; Glade exists after last. Previously it would exit after first encountered.

- bugfix: OASIS Ctrapezoid tpe 3 read incorrectly.

- bugfix: OASIS handling of paths with equal but non-halfwidth extensions was incorrect.

Version 6.0.11 release on 25/09/2024

- bugfix: OASIS writer error for some unusual array attributes.

- enhancement: OASIS reader now warns on > 32bit integers. Previously it threw an exception.

- bugfix: Verify->Compare (XOR) did not write markers to first cellView.

- enhancement: Allow non-printing chars in export OASIS.

Version 6.0.12 release on 02/10/2024

- bugfix: Export GDS with top cell only outputted cell twice.

- enhancement: Verify->Compare available from cmd line as -xor <file1> <file2> <outfile>, where file1 and file2 are GDS or OASIS files, and outfile is a GDS file containing error markers on gds layer 1001. Note the cmd runs in non-graphics mode, the only output is to the logfile, and Glade exits after the output file is written.

- bugfix: Fix a bug where certain shapes in boolean ops which gave rise to holes weren't reported correctly.

- bugfix: Fix a bug with writing OASIS instances with repetition.

- bugfix: Fix a bug in stipple editor that could cause a crash.

- bugfix: Fix a bug in ui::booleanOp.

- Enhancement: Added a ui::compareCells3 to allow writing shapes to a new lib, and select shapes to be either on original layers or marker layer.

- bugfix: Fix a bug in boolean ops which caused a 'next edge not set...' exception.

Version 6.0.13 release on 16/10/2024

- bugfix: Fix a few bugs when running boolean ops on some bizarre layouts.

Version 6.0.14 release on 4/2/2025

- bugfix: Saving design could trash cellView instances in certain cases.

- enhancement: PCell parameters read from python function arglist and created in layout instances now, including setting default values.
